# 品优购

> Java电商项目 , 模仿京东商城 ; 
> 
>     数据库 : MySql
> 
>     开发工具 : IDEA
> 
>     JDK版本 : 1.8
> 
>     服务器 : Tomcat7
> 
>     框架 : Spring、SpringMVC、MyBatis
> 
>     分布式部署 : Dubbo
> 
>     注册中心 : ZooKeeper
> 
>     缓存技术 : redis
> 
>     Nginx : 反向代理

         - [x] 品牌管理

         - [ ] 商品审核

   - [ ] 广告管理模块

         - [ ] 广告类型管理

         - [ ] 广告管理
